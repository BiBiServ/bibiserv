Achtung!

Bitte darauf achten das der Test ordner :

- nicht verlinkt wird
- keine Seiten enthaelt die u.U. in einer oeffentlichen Installation eine Sicherheitsluecke 
  darstellen koennten

Unter Umstaenden muss der test ordner erst der Datei WEB-INF/classes/filterList.ssv
freigeschaltet werden.


Attention!

Please be sure that the test folder 

- won't be linked 
- doesn't contain pages/code fragments that could be a potential security bug

The filter file WEB-INF/classes/filterList.ssv (de-)activate the test folder.
